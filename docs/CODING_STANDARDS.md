# コーディング規約

## 1. 概要
本ドキュメントでは、本プロジェクトにおけるコーディング規約を定義します。

## 2. 使用言語とバージョン
- Python 3.11以上
- TypeScript 5.0以上

## 3. 開発環境
### 3.1 必須ツール
- Node.js 20.x LTS

### 3.2 推奨IDE
- Visual Studio Code

## 4. コーディングスタイル
### 4.1 命名規則
- 関数名: camelCase
- クラス名: PascalCase
- 定数: UPPER_SNAKE_CASE

### 4.2 インデント
- スペース2個（JavaScript/TypeScript）

## 5. セキュリティ要件
### 5.1 認証・認可
- すべてのAPIエンドポイントに認証を実装

### 5.2 データ保護
- 個人情報は必ず暗号化

## 6. コードレビュー基準
### 6.1 必須チェック項目
- セキュリティ脆弱性の確認
- テストカバレッジ80%以上
